Type Dictionary
---------------

Date: Number (epoc)
MessageIdType: UUID
ThreadIdType: UUID
UserIdType: username string


ThreadInfo Payload
------------------

threadInfo: {
  parentId: ThreadIdType,
  participants: [ UserIdType ],
  threadId: ThreadIdType,
  title: String
}

Message Payload
---------------

message: {
  createdAt: Date
  messageId: MessageIdType,
  text: String,
  threadId: ThreadIdType
  userId: String,
}

User Payload
------------
userInfo: {
  id: String,
  name: String,
  threads: { threadId: dummyValue }
}
